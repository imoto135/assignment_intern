# ETTデータセット カラム説明

## データセット概要

ETT（Electricity Transformer Temperature）は、中国の電力会社から収集された電力用変圧器の温度データセットです。

## カラムの意味

| カラム名 | 正式名称            | 意味                   | 単位 |
| -------- | ------------------- | ---------------------- | ---- |
| **date** | Date                | 日時タイムスタンプ     | -    |
| **HUFL** | High UseFul Load    | 高位有効負荷           | -    |
| **HULL** | High UseLess Load   | 高位無効負荷           | -    |
| **MUFL** | Middle UseFul Load  | 中位有効負荷           | -    |
| **MULL** | Middle UseLess Load | 中位無効負荷           | -    |
| **LUFL** | Low UseFul Load     | 低位有効負荷           | -    |
| **LULL** | Low UseLess Load    | 低位無効負荷           | -    |
| **OT**   | Oil Temperature     | オイル温度（目標変数） | ℃    |

## 詳細説明

### 負荷データ（Load Data）

変圧器には3つのレベルの負荷測定があります:

1. **High（高位）**: 高電圧側の負荷
2. **Middle（中位）**: 中電圧側の負荷
3. **Low（低位）**: 低電圧側の負荷

各レベルで2種類の負荷を測定:

- **UseFul Load（有効負荷）**: 実際に仕事をする電力成分（有効電力）
- **UseLess Load（無効負荷）**: 仕事をしない電力成分（無効電力、リアクティブパワー）

### オイル温度（OT）

- **目的**: 変圧器の健全性を示す最も重要な指標
- **重要性**:
  - 設備劣化の早期発見
  - 事故リスクの予測
  - 保全計画の最適化
- **予測対象**: このデータセットでは**OTを予測すること**が主な目的

## データセットの種類

| データセット | サンプリング間隔 | データ数 | 期間                     |
| ------------ | ---------------- | -------- | ------------------------ |
| **ETTh1**    | 1時間            | 17,420   | 2016-07-01 ～ 2018-06-26 |
| **ETTh2**    | 1時間            | 17,420   | 2016-07-01 ～ 2018-06-26 |
| **ETTm1**    | 15分             | 69,680   | 2016-07-01 ～ 2018-06-26 |
| **ETTm2**    | 15分             | 69,680   | 2016-07-01 ～ 2018-06-26 |

- **h1/h2**: 異なる変圧器からの時間単位データ
- **m1/m2**: 同じ変圧器からの分単位データ（より細かい粒度）

## 電力工学の背景知識

### 有効電力 vs 無効電力

```
┌─────────────────────────────────────┐
│ 有効電力（UseFul Load）              │
│ - 実際に仕事をする電力               │
│ - 照明、モーター、加熱などに使用     │
│ - 電力料金の対象                     │
└─────────────────────────────────────┘

┌─────────────────────────────────────┐
│ 無効電力（UseLess Load）             │
│ - 磁場の形成に使われる電力           │
│ - 変圧器やモーターで必要             │
│ - 送電ロスの原因となる               │
└─────────────────────────────────────┘
```

### なぜオイル温度が重要か？

1. **冷却機能**: 変圧器オイルは内部の熱を冷却
2. **絶縁機能**: 電気的絶縁を提供
3. **劣化指標**: 温度上昇は以下を示唆
   - 過負荷状態
   - 冷却システムの不具合
   - 内部故障の可能性

## PoCでの活用方針

### 予測タスク

**入力**: HUFL, HULL, MUFL, MULL, LUFL, LULL（過去の負荷データ）  
**出力**: OT（将来のオイル温度）

### ビジネス価値

1. **予防保全**: 異常温度を事前に予測し、故障を未然に防ぐ
2. **運用最適化**: 負荷調整により温度を適正範囲に保つ
3. **コスト削減**: 計画的な保全により緊急対応コストを削減

### 評価指標の候補

- **MAE（Mean Absolute Error）**: 平均絶対誤差
- **RMSE（Root Mean Square Error）**: 二乗平均平方根誤差
- **MAPE（Mean Absolute Percentage Error）**: 平均絶対パーセント誤差
